Инструкция по использованию бота недвижимости
Добро пожаловать в бот поиска недвижимости.
Для защиты от автоматизированных запросов и обеспечения комфортной работы всем пользователям реализована простая математическая проверка (капча).
Порядок работы с ботом:

Запуск бота
Нажмите кнопку «Start» или отправьте команду /start.
Прохождение проверки (капча)
Вам будет предложен простой математический пример.
При правильном ответе с первой или второй попытки — доступ открывается мгновенно.
При второй подряд ошибке — временная блокировка на 2 минуты.
По истечении 2 минут вы автоматически сможете пройти проверку заново.
Перезапуск бота или удаление диалога не сокращают время блокировки.

Главное меню
После успешного прохождения проверки открывается основное меню со следующими разделами:- Квартиры
│   - 1-комнатные
│   - 2-комнатные
│   - 3-комнатные
│   - 4-комнатные
│   - 5+-комнатные
─ Дома и коттеджи
├─ Новостройки
├─ Земельные участки
└─ Коммерческая недвижимость
Поиск объектов
Выберите нужный раздел и подраздел.
Бот покажет актуальные объявления с сайта в удобном формате (фото, цена, основные характеристики, ссылка на полное объявление).
При необходимости вы можете запросить дополнительные фильтры или обновить список.


Важные замечания:

Все данные берутся напрямую с сайта недвижимости в режиме реального времени.
Бот не хранит историю ваших запросов и не требует регистрации.
При превышении лимитов сайта может потребоваться повторное прохождение капчи (это нормально и занимает несколько секунд).
